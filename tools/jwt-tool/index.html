JWT解析

JWT 解析在线工具，支持本地解码 header 与 payload、查看声明字段和过期时间，适合认证联调与 token 排查。

工具介绍

JWT 解析工具用于在浏览器里快速查看 token 的 header、payload 和过期时间，方便排查认证流程中的声明字段与上下文信息。

当前页面侧重本地解码与可读化展示，不会验证签名，也不会重新生成或签发 token，适合调试时先快速确认内容是否正确。

适用场景

  • 检查登录或接口联调中的 JWT，确认 issuer、subject、audience、role 等声明字段是否符合预期。
  • 在排查鉴权问题时快速查看 exp 等时间字段，判断 token 是否过期或上下文是否写错。
  • 把 token 内容留在本地浏览器里查看，减少将测试凭据粘贴到第三方平台的需要。

使用步骤

  1. 粘贴完整 JWT 字符串，通常为 `header.payload.signature` 格式。
  2. 点击“解析 JWT”，查看格式化后的 header、payload 和过期时间字段。
  3. 按需复制结果，继续结合 JSON、Base64 或哈希工具完成后续排查。

常见问题

JWT解析 会上传我的 token 吗？

除非页面明确说明需要网络请求，当前解析逻辑默认只在浏览器本地运行。

JWT解析 会校验签名吗？

当前工具主要用于解码与查看可读内容，不会验证签名，也不会重新签发 JWT。

什么时候适合使用 JWT解析？

当你需要联调登录态、核对声明字段、检查 payload 内容或确认 token 是否过期时，这个工具最合适。
