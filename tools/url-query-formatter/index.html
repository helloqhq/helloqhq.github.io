URL GET参数格式化

URL GET 参数格式化在线工具，支持粘贴完整 URL 或 query string，本地解析、解码、统计重复参数并输出可复制结果。

工具介绍

URL GET 参数格式化工具用于把完整 URL、? 开头的查询串或裸 query string 整理成可读表格和结构化 JSON。

它会保留重复参数，显示原始值与解码值，并提供紧凑 query string 输出，适合接口联调、埋点排查和跳转链接核对。

适用场景

  • 整理带有大量 GET 参数的接口地址，快速确认参数名、参数值和重复项。
  • 排查跳转链接、回调地址或埋点 URL 中的百分号编码内容是否符合预期。
  • 把临时 query string 转成结构化结果，再继续复制、对比或写入调试记录。

使用步骤

  1. 粘贴完整 URL、? 开头的查询串，或直接粘贴 a=1&b=2 形式的 GET 参数。
  2. 点击“格式化”查看参数表格、重复项统计和结构化 JSON 输出。
  3. 按需使用“压缩输出”生成紧凑 query string，或复制当前结果进入下一步调试。

常见问题

URL GET参数格式化 会上传我的数据吗？

除非页面明确说明需要网络请求，当前主要处理流程默认只在浏览器本地运行。

什么时候适合使用 URL GET参数格式化？

当你需要为联调前检查、可读化查看和结果复制快速得到整理后的输出结果时，URL GET参数格式化 会比较合适。

使用 URL GET参数格式化 前需要准备什么？

通常只需要准备好原始文本或结构化片段，并明确下一步想检查、复制或继续使用的结果。
