Privacy Policy

Read the privacy policy and local-processing statement for UTool.

Welcome to UTool

UTool values user privacy. This page explains what site-level access data may be processed and how tool interactions are designed to stay in the browser whenever possible.

1. Local-first processing

Most tools run locally in the browser. Text, images and generated results are intended to stay on the client side by default.

  • Conversions, formatting, calculations and generators are performed locally when possible
  • We do not intentionally store the business data entered into the tools
  • You should still avoid using shared devices for highly sensitive data

2. Basic access data

The site may collect anonymous analytics such as page visits, browser type, viewport size and error logs to improve usability and stability.

3. Cookies and local storage

Browser storage may be used for language preferences, UI state and recent tool context so the site feels consistent between visits.

4. Third-party resources

If the site integrates third-party monitoring, hosting providers or external links, those services may process access data under their own policies.

5. Contact

If you have any privacy questions, please use the Contact page.
